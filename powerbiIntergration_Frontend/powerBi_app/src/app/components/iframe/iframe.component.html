Publication sur le web à partir de Power BI

Prérequis

Créer des codes d'intégration avec la fonction Publier sur le web

Avec l'option "Publier sur le web" de Power BI, vous pouvez facilement intégrer du contenu interactif Power BI dans des articles de blog, des sites web, des e-mails ou des médias sociaux. Vous pouvez également facilement modifier, mettre à jour, actualiser ou arrêter le partage de vos visualisations publiées.

Avertissement

Lorsque vous utilisez Publier sur le web, n'importe qui sur Internet peut consulter votre rapport ou votre visualisation publiée. La consultation ne nécessite aucune authentification. Cela inclut la visualisation de données détaillées agrégées dans vos rapports. Avant de publier un rapport, assurez-vous qu'il est approprié de partager publiquement les données et les visualisations. Ne publiez pas d'informations confidentielles ou exclusives. En cas de doute, vérifiez les politiques de votre organisation avant de publier.

Prérequis

- Vous avez besoin d'une licence Microsoft Power BI pour publier sur le web depuis votre espace de travail.
- Vous avez besoin d'une licence Microsoft Power BI Pro pour publier sur le web depuis les espaces de travail de groupe.
- La fonction Publier sur le web est disponible pour les rapports que vous pouvez modifier dans vos espaces de travail personnels et de groupe.

Créer des codes d'intégration avec la fonction Publier sur le web

Suivez ces étapes pour utiliser la fonction Publier sur le web. Consultez l'avertissement plus tôt dans cet article avant de publier sur le web.

  1. Ouvrez un rapport dans un espace de travail que vous pouvez modifier, puis sélectionnez Fichier > Intégrer le rapport > Publier sur le web (public).
  2. Publier sur le web sur Plus d'options
  3. Si votre administrateur Power BI ne vous a pas autorisé à créer des codes d'intégration, vous devrez peut-être le contacter.
